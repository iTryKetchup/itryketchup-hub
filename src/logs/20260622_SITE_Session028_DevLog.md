## 3. Game Snapshot

Kitchen Wars: The Door v1.0 proved the core loop: switch lanes, shoot incoming threats, jump low obstacles, duck high obstacles, build a score, and restart for another run.

Version 1.1 keeps that loop intact and makes the game read more clearly as a tiny kitchen-chaos escape. The player is now a rogue ketchup bottle sprinting for freedom down a refrigerator door shelf, with themed hazards sliding and rolling into the lanes.

The current playable version includes:

- ketchup bottle player art
- fridge-door shelf environment
- butter jump obstacle
- rolling soda-can enemy
- overhanging wire-rack obstacle
- cleaned-up local browser leaderboard
- desktop keyboard controls
- mobile/touch controls

## 4. What Worked

- Touch support makes the game more playable on phones and tablets.
