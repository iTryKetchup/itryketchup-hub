## 5. Decisions Made

| Decision | Why |
|----------|-----|
| Treat this milestone as Website v1.3. | It added interactive polish, not just static visual polish. |
| Keep the terminal as a lightweight Easter egg. | It adds personality without becoming a full fake shell or app. |
| Use attached hover buffs instead of complex cursor-following effects. | Keeps the interaction simpler, clearer, and less fragile. |
| Use Asset Loading / Pipeline Pending blocks as building-in-public design language. | Unfinished areas stay honest without implying fake launches. |
| Keep YouTube, games, mascot animation, and playable builds marked as pending / not public. | Prevents the homepage from claiming work that is not live yet. |
| Keep Dev Logs separate from implementation work. | The log documents the work; it does not become the implementation step. |
