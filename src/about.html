About the studio

What iTryKetchup Studio is

iTryKetchup Studio is a public-facing creative studio identity for small games, web projects, dev logs, and practical software experiments.

View Projects Read Dev Logs Contact

What the studio builds

The studio collects projects that are playful enough to have personality and organized enough to ship. Some ideas are games, some are tools, some are fan projects, and some are early planning notes waiting for their turn.

Why this site exists

This hub gives the studio one clear public home instead of scattered links, unfinished placeholders, or project notes spread across different places. It makes progress easier to find, easier to share, and easier to maintain.

Current direction

Right now, the site is preparing the foundation for future updates. Finished work will be shown honestly. Planned work will be marked clearly. Anything not ready yet gets a proper Coming Soon or Parking Lot spot instead of a broken promise.

Build philosophy

  • Function first.
    Simple, working pages beat a giant system that never launches.
  • Clear states.
    Active, planned, coming soon, archive, disabled, and fan project labels keep expectations honest.
  • Maintenance matters.
    Shared data files make project cards and dev logs easier to update later.
