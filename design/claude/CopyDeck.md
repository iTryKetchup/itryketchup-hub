## About Page Copy

**Intro:** iTryKetchup Studio is a public-facing creative studio identity for small games, web projects, dev logs, and practical software experiments.

**What the studio builds:** The studio collects projects that are playful enough to have personality and organized enough to ship. Some ideas are games, some are tools, some are fan projects, and some are early planning notes waiting for their turn.

**Build philosophy:** The hub starts with a static foundation because simple, working pages beat a giant system that never launches. v1.0 is about clarity: clear pages, clear project states, clear contact paths, and clear next steps.

**Current direction:** Right now, the site is preparing the foundation for future updates. Finished work will be shown honestly. Planned work will be marked clearly. Anything not ready yet gets a proper Coming Soon or Parking Lot spot instead of a broken promise.
