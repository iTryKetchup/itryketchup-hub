# Design References

This folder stores raw design references and mockups for iTryKetchup Studio Hub.

Files here are not deployed directly. Claude/DC mockups may contain preview-only syntax such as `x-dc`, `sc-for`, `sc-if`, `support.js`, or `{{ template variables }}`.

Production code must live in `src\` only. Do not copy `x-dc`, `sc-for`, `sc-if`, `support.js`, or `{{ template variables }}` into `src`.

Use these references for layout, visual direction, and data architecture ideas only.
